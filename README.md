# Project Tribulation1

## Run with Rojo

1. Start the dev server: `rojo serve default.project.json`
2. In Roblox Studio, open the Rojo plugin and connect to `localhost:34872`

## Starter Scripts Added

- Shared config: `src/ReplicatedStorage/Shared/GameConfig.lua`
- Shared remote names: `src/ReplicatedStorage/Shared/RemoteNames.lua`
- Server bootstrap: `src/ServerScriptService/Bootstrap.server.lua`
- Client bootstrap: `src/StarterPlayer/StarterPlayerScripts/Bootstrap.client.lua`
